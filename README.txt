Завдання 1
Скласти опис класу для представлення часу. Передбачити 
можливість встановлення часу та зміни його окремих полів (година, 
хвилина, секунда) з перевіркою допустимості введених значень. У разі 
наявності неприпустимих значень та полів викидаються винятки. Створити 
метод зміни часу на задану кількість годин, хвилин і секунд.
Описати клас – спадкоємець, застосувати абстрактні методи /класи
/властивості, віртуальні методи /властивості.
Написати програму, яка демонструвала б усі розроблені елементи
класу.
Алгоритм:
1. Заповнення полів класу Час
2. Перевірка на допустимість введених значень
3. Виведення отриманих даних
4. Використання методу для додавання часу до початкових значень
5. Виведення нових даних
